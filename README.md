# 🏋️ 스포츠지도사 문제은행 앱

2급 전문스포츠지도사 필기시험 대비 문제은행 웹 앱입니다.

## ✅ 구현된 기능

### 🏠 홈 탭
- **학습 통계 대시보드**: 전체 문제 수, 풀이 수, 정답률, 오답 수 표시
- **연도 선택**: 2016~2025년(2024 제외) 9개년도 기출문제 지원
- **과목 선택**: 연도별 과목 리스트, 각 과목별 풀이 현황 표시
- **최근 학습 기록**: 최근 학습한 과목 빠른 접근

### 📝 문제 탭
- **문제 풀기**: 과목별 문제를 순서대로 풀기
- **즉시 정답 확인**: 선택 시 바로 정답/오답 표시 (설정에서 변경 가능)
- **상세 해설**: 정답 해설 + 오답 해설 제공
- **진행률 표시**: 프로그레스 바로 풀이 진행 현황 파악
- **문제 목록 그리드**: 번호 그리드로 문제 이동 (정답/오답 색상 구분)
- **문제 섞기**: 랜덤 순서로 문제 풀기
- **결과 화면**: 풀이 완료 후 정답률 원형 차트, 통계, 다시풀기 기능
- **연도/과목 전환**: 퀴즈 화면에서 바로 연도/과목 변경 가능

### 📖 오답노트 탭
- **자동 오답 수집**: 틀린 문제 자동 저장
- **과목별 필터링**: 전체 또는 과목별 오답 필터
- **오답 상세 보기**: 내 답, 정답, 해설 모두 확인
- **다시풀기**: 오답에서 바로 해당 과목으로 이동
- **개별/전체 삭제**: 오답 관리 기능
- **오답 수 배지**: 하단 네비게이션에 오답 개수 표시

### 📌 요점정리 탭
- 플레이스홀더 (추후 업데이트 예정)

### ⚙️ 설정 탭
- **즉시 정답 확인**: ON/OFF
- **해설 자동 표시**: ON/OFF  
- **보기 순서 섞기**: 선택지 순서 랜덤화
- **학습 기록 초기화**: 모든 데이터 초기화

## 📊 데이터 구조

### 시험 데이터 (JSON 파일)
각 연도별 JSON 파일은 과목별 문제, 선택지, 정답, 해설을 포함합니다.

### 로컬 스토리지
- `sportsQuiz_wrong` - 오답노트 데이터
- `sportsQuiz_history` - 최근 학습 기록
- `sportsQuiz_settings` - 사용자 설정
- `sportsQuiz_solved` - 풀이 기록 (정답/오답)
