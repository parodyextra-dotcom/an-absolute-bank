# 스포츠지도사 문제은행

2급 스포츠지도사 필기시험 대비 학습 웹앱 (2016~2025년 기출문제)

## 주요 기능

### ✅ 구현 완료
- **홈**: 학습 현황 대시보드, D-Day 카운트다운, 일일 문제, 최근 학습 기록
- **요점정리**: 과목별 핵심 요약 + 관련 기출문제 **풀기 모달** (선택지 클릭으로 실제 풀이 가능, 정답률 표시)
- **문제풀기**: 연도별/과목별 문제풀이, 실전 랜덤 모의고사 (타이머 지원)
- **오답노트**: 틀린 문제 복습, 북마크 관리
- **설정**: 과목 선택 (7과목 중 5과목 **모달창으로 변경 가능**), 학습 설정, 데이터 초기화
- **글자 크기 조절**: 요점정리 & 문제풀기 화면에서만 표시 (80%~140%)
- **D-Day 설정**: 시험일 카운트다운

### 데이터
- 2016~2025년 (10개년) 기출문제 JSON 데이터
- 7개 과목: 스포츠사회학, 스포츠교육학, 스포츠심리학, 한국체육사, 운동생리학, 운동역학, 스포츠윤리

## 파일 구조
```

data/
  ├── 2016.json      # 2016년 기출문제 데이터

  ├── 2019.json

  ├── 2021.json

  ├── 2023.json

  └── 2025.json
```

## 접속 경로
- **메인 페이지**: `index.html`
- 하단 네비게이션: 홈 / 요점정리 / 문제풀기 / 오답노트 / 설정

## 최근 업데이트 내역
1. **과목 선택 모달**: 설정에서 과목 선택 시 모달창이 열려 선택/해제 후 저장 가능
2. **글자 크기 컨트롤**: 요점정리·문제풀이 화면에서만 표시 (홈, 오답노트, 설정에서는 숨김)
3. **관련 기출문제 풀기**: 요점정리의 관련 기출문제를 전체 모달에서 직접 풀 수 있음 (선택지 클릭 → 정답/오답 표시 → 해설 보기 → 전체 정답률 표시)

## 기술 스택
- HTML5 / CSS3 / Vanilla JavaScript
- 외부: Google Fonts (Noto Sans KR), Font Awesome 6.5
- 데이터 저장: localStorage
