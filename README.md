# 스포츠지도사 문제은행

## 프로젝트 개요
2급 전문스포츠지도사 필기시험 문제은행 웹 애플리케이션입니다.
2016년~2025년도 기출문제를 학습하고 풀어볼 수 있습니다.

## 구조
- `index.html` - 메인 웹 애플리케이션 (SPA)
- `data/` - 연도별 시험 문제 JSON 데이터
  - `2016.json.txt` ~ `2025.json.txt`
