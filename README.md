# 🏋️ 스포츠지도사 문제은행

2급 전문스포츠지도사 필기시험 대비 문제은행 웹 애플리케이션

## 📌 프로젝트 개요
- **목적**: 2급 스포츠지도사 필기시험 대비 학습 도구
- **과목**: 스포츠사회학, 스포츠교육학, 스포츠심리학, 한국체육사, 운동생리학, 운동역학, 스포츠윤리 (7과목)
- **기출 범위**: 2016~2025년 (10개년)

## 🗂️ 주요 기능

### 1. 메인 문제은행 (`index.html`)
- 과목별 문제 풀이
- 연도별 기출문제 풀이
- 요점정리 학습
- 학습 진도 관리

### 2. ⚔️ 배틀모드 (`battle.html`)
- Firebase 기반 실시간 멀티플레이어 퀴즈 배틀
- 방 생성/참가 (방 코드 공유)
- 과목 선택 → 문제 수 선택 → 실시간 대전
- **한국체육사**: 기출문제(200문제) + 추가문제(300문제) = **총 500문제 풀**에서 랜덤 출제

## 📁 파일 구조

## ✅ 구현 완료 기능

1. ✅ 메인 문제은행 (index.html) - 과목별/연도별 풀이
2. ✅ 배틀모드 (battle.html) - 실시간 멀티플레이어
3. ✅ **한국체육사 배틀모드 확장**: 기출문제 + 한국체육사300 추가문제 랜덤 혼합 출제
   - 기출문제: 2016~2025년 한국체육사 과목 약 200문제
   - 추가문제: 한국체육사300.json에서 300문제
   - 배틀모드에서 한국체육사 선택 시 총 500문제 풀에서 랜덤 셔플 후 출제

## 🔧 기술 변경 내역 (배틀모드 한국체육사 확장)

### 수정 파일: `battle.html`
1. `EXTRA_DATA_URLS` 상수 추가 - 추가 문제 데이터 URL 관리
2. `extraData` 상태 변수 추가 - 추가 문제 데이터 저장
3. `loadAllData()` 함수 수정 - 기출 데이터와 추가 데이터를 병렬 로드
4. `normalizeExtraData()` 함수 추가 - 한국체육사300 JSON 형식을 배틀 문제 형식으로 변환
5. `generateBattleQuestions()` 함수 수정 - 기출 + 추가문제를 합쳐서 랜덤 셔플

### 신규 파일: `data/한국체육사300.json`
- `한국체육사300.json.txt`에서 마크다운 펜스 제거, 잘못된 제어문자 수정하여 클린 JSON 생성
