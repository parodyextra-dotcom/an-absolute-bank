# 스포츠지도사 문제은행

2급 전문스포츠지도사 필기시험 대비 문제은행 웹 애플리케이션

## ✅ 구현된 기능

### 메인 기능
- **기출문제 풀이**: 2016~2025년 기출문제 과목별/연도별 학습
- **요점정리**: 과목별 핵심 요약, 빈출 TOP 20, 단원별 주제 학습
- **동영상강의**: 요점정리 메뉴에서 과목별 YouTube 강의 목차 제공 (펼치기/접기)
- **모의고사**: 실전 시험 환경 모의고사
- **배틀모드**: 대전형 학습 모드 (battle.html)
- **한국체육사 300제**: 별도 문제은행

### 동영상강의 (7개 과목)

### UI/UX
- PWA 지원 (manifest.json, 오프라인 가능)
- 모바일 최적화 반응형 디자인
- 과목 선택/설정 기능
- 학습 진행률 추적
- 다크모드 지원

## 📂 파일 구조

```
index.html              - 메인 페이지 (문제풀이, 요점정리, 모의고사)
battle.html             - 배틀모드 페이지
manifest.json           - PWA 매니페스트
images/
  └── knsu-mascot.png   - 마스코트 이미지 (아이콘)
data/
  ├── 2016.json         - 2016년 기출문제
  ├── 2017.json         - 2017년 기출문제
  ├── 2018.json         - 2018년 기출문제
  ├── 2019.json         - 2019년 기출문제
  ├── 2020.json         - 2020년 기출문제
  ├── 2021.json         - 2021년 기출문제
  ├── 2022.json         - 2022년 기출문제
  ├── 2023.json         - 2023년 기출문제
  ├── 2024.json         - 2024년 기출문제

```

## 🔗 주요 URI

| 경로 | 설명 |
|------|------|
| `index.html` | 메인 페이지 (기출문제, 요점정리, 모의고사) |
| `battle.html` | 배틀모드 |

## 🛠 기술 스택

- HTML5, CSS3, JavaScript (Vanilla)
- Noto Sans KR (Google Fonts)
- Font Awesome 6.5 (아이콘)
- PWA (Progressive Web App)

- 서비스워커(sw.js) 추가로 완전한 오프라인 지원
- 과목별 오답 노트 기능 강화
- 학습 통계 대시보드 개선
- 추가 연도 기출문제 데이터 업데이트
