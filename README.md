# 스포츠지도사 문제은행 (Sports Coach Exam Question Bank)

## 프로젝트 개요
2급류 체육지도자 필기시험 기출문제를 학습할 수 있는 웹 기반 문제은행 애플리케이션입니다.  
2016년부터 2025년까지의 기출문제를 과목별로 풀고, 해설을 확인하며, 오답노트와 북마크 기능을 활용할 수 있습니다.

## 주요 기능

### ✅ 완료된 기능
- **연도별 기출문제 풀기** (2016~2025, 총 10개년)
- **7개 과목별 학습**: 스포츠사회학, 스포츠교육학, 스포츠심리학, 한국체육사, 운동생리학, 운동역학, 스포츠윤리
- **즉시 채점 모드**: 문제 풀이 즉시 정답/오답 확인
- **상세 해설**: 정답 해설 + 오답 이유 설명
- **보기(〈보기〉) 자동 표시**: 문제에 포함된 보기 내용 자동 추출 및 정리
- **오답노트**: 틀린 문제 자동 저장 및 복습
- **북마크**: 중요 문제 저장
- **학습 통계**: 과목별 정답률, 풀이 기록 추적
- **랜덤 출제**: 과목 혼합 랜덤 문제 풀기
- **보기 순서 섞기**: 선택지 순서 무작위 배치
- **모바일 반응형**: 하단 네비게이션 바 기반 UI
- **D-day 설정**: 시험일까지 남은 일수 표시
- **글꼴 크기 조절**: 가독성 맞춤 설정

### 데이터 수정 사항
- **2019.json.txt**: JSON 파싱 오류 2건 수정 (문자열 내 제어 문자)
- **2025.json.txt**: 보기 필드 66개 추가 (fix3.html로 적용)
- **런타임 보기 추출**: 170개 문항의 보기 내용을 질문 텍스트에서 자동 추출
- **BOGI_SUPPLEMENT DB**: 자동 추출 불가능한 43개 문항의 보기 데이터 수동 보완

## 보기(〈보기〉) 처리 로직
데이터 파일의 보기 내용은 3가지 방식으로 처리됩니다:

1. **별도 필드**: `"보기": "..."` 필드가 있으면 그대로 사용 (2024, 2025)
2. **자동 추출**: 질문 텍스트에 `[보기: ...]`, `\n\n`, 또는 `\n` 구분자로 포함된 보기 내용을 런타임에 분리
3. **보충 DB**: 자동 추출 불가능한 43개 문항은 `BOGI_SUPPLEMENT` 객체에서 제공
