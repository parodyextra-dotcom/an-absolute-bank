# 🏋️ 스포츠지도사 문제은행

2급 전문스포츠지도사 필기시험 대비 문제은행 웹 애플리케이션

## ✅ 구현된 기능

### 핵심 기능
- **기출문제 풀이**: 2016~2025년 기출문제 (연도별/과목별 선택)
- **7개 과목 지원**: 스포츠사회학, 스포츠교육학, 스포츠심리학, 한국체육사, 운동생리학, 운동역학, 스포츠윤리
- **정답 해설**: 정답 및 오답 해설 제공
- **요점정리**: 전과목 핵심 요약 정리
- **학습 진도 관리**: 과목별/연도별 학습 진행률 추적

### 매일 10문제 풀기 - 정답/오답 애니메이션
- **정답 시**: 🏋️ "영~차!" 역도 캐릭터가 바벨을 들어올리는 이미지 + 축하 파티클 애니메이션
- **오답 시**: 😢 "돌대가리!" 역도 캐릭터가 주저앉은 이미지 + 충격파 애니메이션
- 줌 인/아웃 + 반짝임/흔들림 등 역동적인 시각 효과

### PWA / 홈화면 추가
- **홈화면 아이콘**: KNSU 마스코트 이미지 (한국체육대학교 역도선수 캐릭터)
- **Web App Manifest**: `manifest.json` - standalone 모드 지원
- **Apple Touch Icon**: iOS 홈화면 추가 시 전용 아이콘 표시
- **Favicon**: 브라우저 탭 아이콘

### 로딩 화면
- KNSU 마스코트 이미지 + 역기(덤벨) 아이콘 애니메이션
- 보라색 그라디언트 배경
- 로딩 프로그레스 바

## 📱 홈화면 추가 방법

### iOS (Safari)
1. Safari에서 사이트 열기
2. 공유 버튼 탭 → "홈 화면에 추가"
3. KNSU 마스코트 아이콘이 홈화면에 추가됨

### Android (Chrome)
1. Chrome에서 사이트 열기
2. 메뉴(⋮) → "홈 화면에 추가" 또는 자동 설치 배너
3. 앱처럼 사용 가능
