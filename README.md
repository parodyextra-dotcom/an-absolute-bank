# 스포츠지도사 문제은행

2급 전문스포츠지도사 필기시험 문제은행 웹앱입니다.

## 주요 기능

- **매일 10문제 풀기**: 매일 랜덤으로 10문제를 풀 수 있는 모드
  - ✅ **정답 시 성공 애니메이션**: KNSU 역도 캐릭터가 바벨을 들어올리는 "영~차!" 이미지가 바운스 + 빛나는 효과 + 색종이 파티클과 함께 등장
  - ❌ **오답 시 실패 애니메이션**: KNSU 역도 캐릭터가 지쳐 앉아있는 "돌대가리" 이미지가 드롭 + 흔들림 + 충격선 효과와 함께 등장
  - 터치/클릭으로 애니메이션 조기 닫기 가능
- **과목별 풀기**: 과목별로 문제를 풀 수 있는 모드
- **모의고사**: 실제 시험처럼 풀 수 있는 모드
- **오답 노트**: 틀린 문제를 자동으로 저장하여 복습
- **북마크**: 중요한 문제를 북마크하여 관리
- **통계**: 학습 진행률 및 정답률 확인

## 구현된 애니메이션 상세

### 정답 애니메이션
1. 녹색 그라데이션 배경 오버레이
2. 캐릭터 이미지 바운스 인 (scale + rotate)
3. 빛나는 글로우 펄스 효과
4. 14개의 금색 스파클 파티클 방사
5. 20개의 다색 색종이 폭발 효과
6. "🎉 정답! 영~차!" 텍스트 페이드인
7. 1.4초 후 자동 사라짐

### 오답 애니메이션
1. 빨간색 그라데이션 배경 오버레이
2. 캐릭터 이미지 드롭 인 (위에서 떨어짐)
3. 좌우 흔들림(shake) 효과
4. 10개의 빨간 충격선 방사
5. "😢 오답... 돌대가리!" 텍스트 페이드인
6. 1.6초 후 자동 사라짐

## 기술 스택

- Google Fonts (Noto Sans KR)

## 개발 참고

- 애니메이션은 `매일 10문제 풀기` 모드에서만 작동합니다

- 오버레이 클릭 시 조기 닫기 지원
