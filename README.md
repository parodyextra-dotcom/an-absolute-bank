# 🏋️ 스포츠지도사 문제은행

2급 전문스포츠지도사 필기시험 대비 웹앱 (2016~2025년)

## 🎯 프로젝트 목표
- 2급 스포츠지도사 필기시험 기출문제를 효과적으로 학습할 수 있는 모바일 최적화 웹앱
- 과목별 요점정리를 통한 핵심 이론 학습 지원

## ✅ 구현된 기능

### 1. 홈 (메인 페이지)
- D-Day 카운트다운 (시험일까지 남은 일수)
- 학습 통계 (전체 문제, 풀은 문제, 정답률, 오답 수)
- 연도별(2016~2025) 과목 선택
- 최근 학습 이력

### 2. 요점정리 📖 (NEW)
- **과목별 페이지**: 7개 과목 카드 형태 표시 (스포츠교육학, 스포츠사회학, 스포츠심리학, 스포츠윤리, 운동생리학, 운동역학, 한국체육사)
- **단원별 페이지**: 과목 선택 시 해당 과목의 대단원 목록 표시
- **소단원별 페이지**: 단원 선택 시 소단원 목록 표시
- **요점 상세 페이지**: 소단원 선택 시 핵심 내용 + 관련 기출문제 연결
- 브레드크럼 내비게이션으로 계층 이동
- 이전/다음 소단원 탐색 버튼
- 기출문제에서 관련 키워드 자동 매칭

### 3. 문제풀기
- 연도·과목별 문제 풀이
- 즉시 정답 확인 & 해설 표시
- 문제 섞기 기능
- 문제 번호 그리드 (전체 진행 상황 파악)
- 결과 화면 (정답률 원형 차트)
- 북마크 기능

### 4. 오답노트
- 틀린 문제 자동 저장
- 과목별 필터링
- 오답 상세 보기 (해설 포함)
- 다시풀기 기능
- 북마크 탭 (북마크한 문제 모아보기)

### 5. 설정
- 즉시 정답 확인 ON/OFF
- 해설 자동 표시 ON/OFF
- 보기 순서 섞기 ON/OFF
- 학습 기록 초기화
- D-Day 설정

## 🔗 URI / 페이지 구조

| 페이지 | 설명 | 네비게이션 |
|--------|------|-----------|
| `#pageHome` | 홈 (기본 페이지) | 하단 메뉴 "홈" |
| `#pageSummary` | 요점정리 | 하단 메뉴 "요점정리" |
| `#pageQuiz` | 문제풀기 | 하단 메뉴 "문제풀기" |
| `#pageWrong` | 오답노트/북마크 | 하단 메뉴 "오답노트" |
| `#pageSettings` | 설정 | 하단 메뉴 "설정" |

### 요점정리 계층 구조
```
전체 과목 (7개)
  └── 과목별 단원 목록
       └── 단원별 소단원 목록
            └── 소단원 상세 (핵심 포인트 + 관련 기출문제)
```
