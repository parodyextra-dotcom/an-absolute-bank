# 🏋️ 스포츠지도사 문제은행

2급 전문스포츠지도사 필기시험 대비 웹앱 (2016~2025년)

## 🎯 프로젝트 목표
- 2급 스포츠지도사 필기시험 기출문제를 효과적으로 학습할 수 있는 모바일 최적화 웹앱
- 과목별 요점정리를 통한 핵심 이론 학습 지원
- 매일 10문제, 단원별, 모의고사 등 다양한 문제풀기 모드 제공

## ✅ 구현된 기능

### 1. 홈 (메인 페이지)
- **D-Day 카운트다운** (시험일까지 남은 일수, 설정 가능)
- **매일 10문제 풀기 박스** - 오늘의 진행률 표시, 클릭 시 문제풀기로 이동
- **단원별 문제풀기 박스** - 전체 문제 풀이 진행률 표시
- **연도별 모의고사 박스** - 연도별 완료 현황 표시
- **학습 통계 카드** (전체 문제, 풀은 문제, 정답률, 오답 수)
- **최근 학습 이력**

### 2. 요점정리 📖
- **과목별 페이지**: 7개 과목 카드 형태 표시
- **단원별 페이지**: 대단원 목록 표시
- **소단원별 페이지**: 소단원 목록 및 상세 핵심 내용
- **관련 기출문제 연결**: 키워드 매칭으로 기출문제 자동 연결
- 브레드크럼 내비게이션

### 3. 문제풀기 (3가지 모드) ✏️
- **매일 10문제**: 날짜 기반 랜덤 10문제 출제, 연속 학습일 추적, 히스토리 기록
- **단원별 문제풀기**: 과목별 확장 패널, 연도별 문제풀기, 원형 진행률 표시
- **연도별 모의고사**: 연도별 전 과목 현황, 과목 선택 또는 전 과목 순서대로 풀기
- 탭 기반 허브 UI (매일 10문제 / 단원별 / 모의고사)
- 문제 풀기 화면: 즉시 정답확인, 해설, 북마크, 문제 그리드, 섞기

### 4. 오답노트
- 틀린 문제 자동 저장 / 과목별 필터 / 상세 해설
- 다시풀기 / 북마크 탭

### 5. 설정
- 즉시 정답 확인 / 해설 자동 표시 / 보기 순서 섞기
- 학습 기록 초기화 (매일10문제 기록 포함)
- D-Day 설정

## 📁 파일 구조

```
index.html              # 메인 앱 (SPA, 단일 파일)
data/
  ├── 2016.json ~ 2025.json   # 연도별 기출문제 데이터
README.md               # 프로젝트 문서
```

## 🔗 페이지 구조

| 페이지 | 설명 | 네비게이션 |
|--------|------|-----------|
| `#pageHome` | 홈 (매일10문제/단원별/모의고사 진행률 + 통계) | 하단 "홈" |
| `#pageSummary` | 요점정리 (과목→단원→소단원) | 하단 "요점정리" |
| `#pageQuiz` | 문제풀기 허브 (3탭) + 문제풀이 화면 | 하단 "문제풀기" |
| `#pageWrong` | 오답노트/북마크 | 하단 "오답노트" |
| `#pageSettings` | 설정 | 하단 "설정" |

### 문제풀기 모드
| 모드 | 설명 |
|------|------|
| 매일 10문제 | 날짜 시드 랜덤 10문제, 연속 학습일, 히스토리 |
| 단원별 | 과목별 > 연도별로 문제 풀기 |
| 모의고사 | 연도별 전 과목 풀기 또는 과목 선택 |
