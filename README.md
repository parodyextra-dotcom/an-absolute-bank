# 🏋️ 스포츠지도사 문제은행

2급 전문스포츠지도사 필기시험 대비 문제은행 웹앱

## 주요 기능

### ✅ 구현 완료
- **홈 화면**: D-Day 카운트다운, 학습 통계(전체문제/풀은문제/정답률/오답수), 연도별 과목 선택
- **문제풀기**: 연도/과목별 문제 풀이, 즉시 정답 확인, 해설 표시, 보기 섞기, 문제 목록(그리드) 보기
- **오답노트**: 틀린 문제 자동 저장 및 복습, 북마크 기능
- **설정**: 즉시 정답 확인, 해설 자동 표시, 보기 순서 섞기, 학습 기록 초기화
- **D-Day 설정**: 시험일 커스텀 설정 가능
- **데이터**: 2016~2025년 기출문제 전체 포함 (2024년 포함)
