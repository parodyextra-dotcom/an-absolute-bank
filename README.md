# 스포츠지도사 문제은행 📚

2급 스포츠지도사 필기시험 대비 문제은행 웹 애플리케이션

## 📌 프로젝트 정보
- **목표**: 2급 스포츠지도사 필기시험 준비를 위한 모바일 최적화 학습 앱
- **시험 범위**: 2016~2025년 기출문제 (10개년)
- **과목**: 스포츠사회학, 스포츠교육학, 스포츠심리학, 한국체육사, 운동생리학, 운동역학, 스포츠윤리 (7과목 중 5과목 선택)

## ✅ 구현 완료 기능

### 핵심 기능
- **홈 대시보드**: D-Day 카운터, 학습 통계, 최근 학습 기록
- **요점정리**: 7과목 핵심 커리큘럼 (단원별/소단원 구조, 관련 기출 연결)
- **문제풀기**: 4가지 모드
  - 🔥 매일 10문제 (일일 랜덤)
  - 📚 단원별 학습 (과목/연도별)
  - 🏆 모의고사 (연도별 전 과목)
  - 🎲 실전 랜덤 모의고사 (전 연도 랜덤 100문제, 100분)
- **오답노트**: 틀린 문제 자동 저장 + 북마크 기능
- **설정**: 과목 선택, 즉시 정답 확인, 자동 다음 문제, 글자 크기 조절

### 최근 업데이트 (v2)
- **미선택 과목 흑백 처리**: 요점정리/문제풀이에서 선택하지 않은 과목은 흑백(grayscale)으로 표시
- **커스텀 다이얼로그**: 미선택 과목 클릭 시 "선택하지 않은 과목입니다. 변경하시겠습니까?" 커스텀 확인 대화상자 표시
  - "아니오" → 대화상자 닫기
  - "예" → 과목 변경 모달 열기
- **타이머 위치 이동**: 모의고사/실전랜덤 남은 시간 타이머가 상단 고정 바에서 과목 제목 바로 아래로 이동
- **선택 과목 연동**: 과목 변경 후 요점정리/문제풀이 화면 즉시 갱신

## 🔗 진입 URI
- `/index.html` — 메인 앱 (모든 기능 포함 SPA)

## 🗄️ 데이터 저장
- **localStorage** 사용
  - `sportsQuiz_selectedSubjects` — 선택된 5과목
  - `sportsQuiz_wrong` — 오답노트
  - `sportsQuiz_bookmarks` — 북마크
  - `sportsQuiz_history` — 학습 기록
  - `sportsQuiz_settings` — 앱 설정
  - `sportsQuiz_solved` — 풀이 현황
  - `sportsQuiz_daily` — 매일 10문제 기록
  - `sportsQuiz_dday` — D-Day 설정
  - `sportsQuiz_fontSize` — 글자 크기
  - `sportsQuiz_randomHistory` — 실전 랜덤 기록
