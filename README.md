# 🏋️ 스포츠지도사 문제은행 v2.0

2급 전문스포츠지도사 필기시험 대비 문제은행 웹 애플리케이션

## 📋 주요 기능

### ✅ 구현된 기능

#### 1. 홈 대시보드
- D-Day 카운트다운 (시험일: **2026년 4월 18일**)
- 전체 학습 통계 (전체 문제, 풀은 문제, 정답률, 오답 수)
- 매일 10문제, 단원별, 모의고사, 실전랜덤 퀵 액세스
- 최근 학습 기록

#### 2. 매일 10문제 풀기 🔥
- 매일 날짜 시드 기반 랜덤 10문제 출제
- 연속 학습 일수(streak) 추적
- **정답 시**: 영~차! 역도 성공 이미지 + 폭죽 애니메이션
- **오답 시**: 돌대가리! 역도 실패 이미지 + 흔들림 애니메이션
- 일별 학습 기록 조회

#### 3. 단원별 문제풀기 📚
- 7과목 × 10개 연도(2016~2025) 문제 학습
- 과목별 진행률 원형 프로그레스 표시
- 연도별 풀이 상태 추적

#### 4. 연도별 모의고사 🏆
- 2016~2025년 10개 연도 모의고사
- **설정에서 선택한 5과목만** 풀기 가능
- 과목당 20문제, 총 100문제
- **100분 타이머** 내장 (5분 이하 시 빨간색 경고)
- 개별 과목 단독 풀기 지원

#### 5. 실전 랜덤 모의고사 🎲 (NEW!)
- 10개 연도에서 **과목별 랜덤 추출**
- 선택 5과목 × 20문제 = 총 100문제
- **100분 타이머** 내장
- 매번 다른 문제 조합으로 실전 감각 향상
- 랜덤 기록 저장

#### 6. 요점정리 📖
- 7과목 전체 커리큘럼 요약
- 단원 > 소단원 > 요점 계층 구조
- **관련 기출문제 전체 모달창** 표시 (문제 + 선택지 + 정답 + 해설)
- 키워드 기반 관련 문제 자동 매칭

#### 7. 오답노트 📝
- 틀린 문제 자동 저장
- 과목별 필터링
- 상세 보기 (문제 + 선택지 + 해설)
- 다시 풀기 / 삭제 기능

#### 8. 북마크 ⭐
- 문제 풀이 중 별 아이콘으로 북마크
- 과목별 필터링
- 상세 보기 모달

#### 9. 글자크기 확대/축소 🔤 (NEW!)
- 화면 우측 상단 +/- 버튼
- 80% ~ 140% 범위 조절
- 설정 자동 저장 (localStorage)

#### 10. 설정 ⚙️
- **선택 과목 설정** (7과목 중 5과목 선택) (NEW!)
- 즉시 정답 확인 ON/OFF
- 해설 자동 표시 ON/OFF
- 보기 순서 섞기 ON/OFF
- 학습 기록 초기화
- D-Day 시험일 설정

## 📁 파일 구조

```
index.html              # 메인 SPA (HTML + CSS + JS)
images/
  ├── correct.png       # 정답 이미지 (역도 성공)
  └── wrong.png         # 오답 이미지 (역도 실패)
data/
  ├── 2016.json         # 2016년 기출문제
  ├── 2017.json         # 2017년 기출문제
  ├── 2018.json         # 2018년 기출문제
  ├── 2019.json         # 2019년 기출문제
  ├── 2020.json         # 2020년 기출문제
  ├── 2021.json         # 2021년 기출문제
  ├── 2022.json         # 2022년 기출문제
  ├── 2023.json         # 2023년 기출문제
  ├── 2024.json         # 2024년 기출문제
  └── 2025.json         # 2025년 기출문제
README.md               # 프로젝트 문서
```

## 🔗 진입 URI

| 경로 | 설명 |
|------|------|
| `index.html` | 메인 앱 (SPA) |
| 하단 네비게이션: 홈 | 대시보드 |
| 하단 네비게이션: 요점정리 | 과목별 요점 + 관련 기출 |
| 하단 네비게이션: 문제풀기 | 4가지 모드 (매일10/단원별/모의고사/실전랜덤) |
| 하단 네비게이션: 오답노트 | 오답 + 북마크 관리 |
| 하단 네비게이션: 설정 | 과목선택 + 학습설정 + 데이터관리 |

### 데이터 저장 (localStorage)
- `sportsQuiz_settings` - 학습 설정
- `sportsQuiz_wrong` - 오답 목록
- `sportsQuiz_history` - 풀이 기록
- `sportsQuiz_solved` - 풀이 완료 데이터
- `sportsQuiz_bookmarks` - 북마크 목록
- `sportsQuiz_dday` - D-Day 설정
- `sportsQuiz_daily` - 매일 10문제 데이터
- `sportsQuiz_fontSize` - 글자 크기 설정
- `sportsQuiz_selectedSubjects` - 선택 과목 (5과목)
- `sportsQuiz_randomHistory` - 실전 랜덤 기록
