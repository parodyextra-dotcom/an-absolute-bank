# 🏋️ 스포츠지도사 문제은행 v2.1

2급 전문스포츠지도사 필기시험 대비를 위한 종합 학습 웹앱입니다.

## ✅ 구현된 기능

### 📖 요점정리 (NEW - 요점정리.txt 연동)
- **첨부된 요점정리.txt 파일**에서 동적으로 파싱하여 렌더링
- **7과목 전체** 요점정리: 스포츠교육학, 스포츠사회학, 스포츠심리학, 스포츠윤리, 운동생리학, 운동역학, 한국체육사
- **형광펜 하이라이트**: 
  - 🟡 노란색: 인용/핵심 용어 (큰따옴표 내용)
  - 🔵 파란색: 영어 학술 용어
  - 🔴 분홍색: 함정 키워드 (아닌 것, 포함되지 않는다 등)
- **빈출도 표시**: 🔥최다빈출(★★★), ⭐빈출(★★), 📌기출(★) 배지
- **과목별 빈출 TOP 20**: 과목 진입 시 빈출 주제 순위표 표시
- **카드형 아코디언 UI**: 주제별 펼치기/접기, 빈출도에 따라 카드 색상 차등
- **관련 기출문제 연동**: 요점정리 주제에서 직접 기출문제 풀이 가능

### ✏️ 문제풀기
- **매일 10문제**: 매일 랜덤 10문제 출제, 연속 학습 기록
- **단원별 문제풀기**: 과목·연도별 집중 학습
- **연도별 모의고사**: 실전처럼 연도별 전 과목 풀기 (100문제, 100분 타이머)
- **실전 랜덤 모의고사**: 10개 연도에서 랜덤 추출 (5과목 × 20문제)
- **즉시 정답 확인/해설 표시** 옵션
- **보기 순서 섞기** 옵션
- **문제 북마크** 기능

### 📝 오답노트 & 북마크
- 틀린 문제 자동 저장
- 과목별 필터링
- 오답 상세 보기 (해설 포함)
- 북마크 관리

### ⚙️ 설정
- 7과목 중 5과목 선택 설정
- 학습 설정 (즉시 정답 확인, 해설 자동 표시, 보기 순서 섞기)
- D-Day 설정
- 학습 기록 초기화
- 글자 크기 조절

### 🏠 홈 대시보드
- D-Day 카운트다운
- 학습 통계 (전체 문제, 풀은 문제, 정답률, 오답 수)
- 학습 진도 표시
- 최근 학습 기록
